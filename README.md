## ✨ Overview

This project is an elegant, highly responsive personal portfolio website designed to showcase professional frontend development skills, creative projects, and technical capabilities. It's built as a modern single-page React application — fast, animated, and fully self-contained on the client side.

### Key Features

| | |
|---|---|
| 🔍 *SEO & Open Graph Ready* | Full search engine optimization configuration and Open Graph tags for rich, polished link previews when shared on social media. |
| 🧩 *Component-Driven Architecture* | A modular interface dynamically injected through a root document framework — clean, reusable, declarative UI building blocks. |
| 🎬 *Fluid Motion Design* | Interactive animations and scroll-based motion physics power smooth, fluid transitions throughout the experience. |
| 📐 *Consistent Iconography* | A standardized inline vector line-icon system keeps every symbol crisp and visually unified across breakpoints. |
| 📬 *Client-Side Contact Pipeline* | An automated inquiry pipeline validates and processes contact form submissions, routing them straight to email — no backend required. |

---

## 🛠️ Tech Stack

| Technology | Role |
|---|---|
| *HTML5* | Semantic document structure, essential search engine metadata, Open Graph link-sharing descriptors, and the baseline script mounting node. |
| *React 19 & React-DOM* | Component-based application architecture, reactive state bindings, and declarative DOM tree rendering. |
| *Tailwind CSS v4 & @tailwindcss/vite* | Full presentation layer via modern utility class layers and a high-performance build-time design engine integration. |
| *Framer Motion* | Complex interactive animation timelines, dynamic entry/exit UI transitions, and smooth element transformations. |
| *Lucide React* | A standardized toolkit of scalable, responsive vector line icons across the interface. |
| *EmailJS-Com* | Intercepts contact form validation events and transmits messages directly via email — no dedicated backend server required. |
| *Vite 8 & Oxlint* | Rapid hot module replacement (HMR) build compilation, paired with high-speed, Rust-based static code analysis for a clean codebase. |

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS recommended)
- npm (bundled with Node.js)

### Installation

bash
# 1. Clone the repository
git clone https://github.com/shayannaqvi45/potfolio-anti.git
cd potfolio-anti

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev


Vite will spin up a local dev server with hot module replacement — open the printed localhost URL in any modern browser to view it live.

### Build for production

bash
npm run build


This compiles the full asset pipeline into an optimized, production-ready bundle.

Or skip the setup entirely and view it live here: 👉 [Live Demo](shayan-portfolio-one.vercel.app)
